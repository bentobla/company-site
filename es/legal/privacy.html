Recopilación de datos en este sitio web

Este sitio web es un sitio estático alojado en GitHub Pages. No recopilamos, almacenamos ni procesamos datos personales a través de este sitio web.

No utilizamos cookies, herramientas de análisis ni mecanismos de seguimiento.

Archivos de registro del servidor

El proveedor de alojamiento (GitHub) puede recopilar y almacenar automáticamente información en archivos de registro del servidor. Esto puede incluir:

  • Tipo y versión del navegador
  • Sistema operativo
  • URL de referencia
  • Nombre de host de la computadora que accede
  • Hora de la solicitud del servidor
  • Dirección IP

Estos datos no se combinan con otras fuentes de datos y se almacenan por GitHub según su política de privacidad.

Contacto

Si nos contacta por correo electrónico, su mensaje y los datos que proporcione (incluida la dirección de correo electrónico y el nombre) se almacenarán con el fin de procesar su consulta y para posibles preguntas de seguimiento. No compartimos estos datos sin su consentimiento.

Sus derechos

  • Solicitar la eliminación de sus datos

  • Oponerse al procesamiento de datos
  • Solicitar la portabilidad de datos
