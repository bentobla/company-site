Raccolta dati su questo sito web

Questo sito web è un sito statico ospitato su GitHub Pages. Non raccogliamo, archiviamo o trattiamo dati personali attraverso questo sito web.

Non utilizziamo cookie, strumenti di analisi o meccanismi di tracciamento.

File di log del server

Il provider di hosting (GitHub) può raccogliere e archiviare automaticamente informazioni nei file di log del server. Ciò può includere:

  • Tipo e versione del browser
  • Sistema operativo
  • URL di riferimento
  • Nome host del computer che accede
  • Ora della richiesta del server
  • Indirizzo IP

Questi dati non vengono combinati con altre fonti di dati e vengono archiviati da GitHub secondo la loro informativa sulla privacy.
