Déclaration de confidentialité pour l'application lœrn

Dernière mise à jour : 5 février 2026

Introduction

Cette déclaration de confidentialité explique comment lœrn (« nous », « notre » ou « l'application ») traite les informations. lœrn est conçu avec la confidentialité comme principe fondamental. Nous croyons que vos données d'apprentissage doivent toujours rester sous votre contrôle.

Sauf lorsque vous choisissez explicitement d'exporter ou de partager du contenu, lœrn ne transmet pas vos données en dehors de votre appareil.

Principes fondamentaux de confidentialité

Conception axée sur la confidentialité

lœrn est construit de zéro pour protéger votre vie privée :

  • Hors ligne par conception : Les fonctionnalités d'apprentissage de base fonctionnent sans connexion Internet
  • Pas de comptes utilisateur : Pas d'inscription, pas de connexion, pas de mots de passe
  • Aucun suivi : Pas d'analyse comportementale, pas de profilage
  • Stockage de données local uniquement : Les données d'apprentissage restent sur votre appareil sauf si vous les exportez ou les partagez explicitement
  • Contrôle de l'utilisateur : Les données quittent votre appareil uniquement par des actions que vous initiez intentionnellement

Données que nous ne collectons pas

Nous ne collectons, stockons ou transmettons PAS :

  • Informations personnelles (nom, adresse e-mail, numéro de téléphone, adresse)
  • Comptes utilisateur ou identifiants d'authentification
  • Contenu d'apprentissage ou matériel d'étude que vous créez ou importez
  • Progrès d'apprentissage, scores ou statistiques
  • Analyses d'utilisation ou données de télémétrie
  • Rapports de crash ou données de diagnostic
  • Données de localisation
  • Listes de contacts ou connexions sociales

Pour les utilisateurs gratuits, les partenaires publicitaires peuvent recevoir des données techniques limitées comme requis pour afficher des publicités (voir la section Publicité ci-dessous).

Stockage des données

Stockage local uniquement

Toutes les données liées à l'apprentissage sont stockées exclusivement sur votre appareil :

  • Paquets d'apprentissage et cartes mémoire
  • Progrès d'étude et statistiques
  • Historique de révision et données de répétition espacée
  • Préférences utilisateur et paramètres
  • Informations de profil local (nom de profil, icône, préférence de langue)

Ces données sont stockées dans une base de données SQLite locale dans le stockage privé de l'application et sont protégées par les mécanismes de sécurité intégrés de votre appareil (iOS Data Protection, Android Keystore).

Pas de synchronisation cloud

lœrn ne fournit pas de synchronisation cloud, de sauvegarde cloud ou de stockage côté serveur. Nous n'exploitons pas de serveurs qui reçoivent ou stockent vos données d'apprentissage.

Connectivité Internet

Noyau d'apprentissage hors ligne

La fonctionnalité d'apprentissage de base de lœrn est entièrement hors ligne et ne nécessite pas d'accès réseau.

Publicité (version gratuite uniquement)

Si vous utilisez la version gratuite de lœrn, l'application affiche des publicités. Seul le module publicitaire accède à Internet afin de :

  • Charger le contenu publicitaire
  • Signaler les impressions et interactions publicitaires
  • Se conformer aux exigences du réseau publicitaire

Le noyau d'apprentissage lui-même ne transmet pas de données d'apprentissage, de contenu ou de progrès.

Partenaires publicitaires

La version gratuite utilise Google AdMob. AdMob peut collecter certaines informations selon sa propre politique de confidentialité, qui peut inclure :

  • Identifiant publicitaire de l'appareil
  • Adresse IP
  • Type d'appareil et système d'exploitation
  • Données d'interaction publicitaire

Pour plus de détails sur les pratiques de données de Google, voir : https://policies.google.com/privacy

Version premium

Si vous achetez la version premium (achat unique) :

  • Toutes les publicités sont supprimées
  • Le module publicitaire n'est pas initialisé
  • L'application ne nécessite pas d'accès Internet pour une utilisation normale
  • Aucune communication réseau n'est effectuée par l'application

Achats intégrés

Traitement des achats

Les achats premium sont traités entièrement par l'Apple App Store ou le Google Play Store.

  • Nous ne collectons ni ne stockons d'informations de paiement
  • Nous ne recevons pas de détails de carte de crédit ou de facturation
  • Nous recevons uniquement la confirmation qu'un achat a été effectué

Le statut premium est stocké localement sur votre appareil. L'achat est lié à votre compte App Store ou Play Store, et non à un compte lœrn.

Importation, exportation et partage

Importation et exportation basées sur des fichiers locaux

lœrn vous permet d'importer et d'exporter des paquets d'apprentissage en utilisant des fichiers JSON ou ZIP :

  • Les fichiers importés sont traités localement sur votre appareil
  • Les fichiers exportés contiennent uniquement du contenu d'apprentissage (questions, réponses, métadonnées)
  • Les fichiers exportés n'incluent pas d'identifiants personnels par défaut

Partage initié par l'utilisateur

Vous pouvez choisir de partager des fichiers exportés en utilisant les fonctionnalités de partage standard de votre appareil.

Clarifications importantes :

  • Les données sont partagées uniquement lorsque vous initiez explicitement le partage
  • lœrn ne transmet pas automatiquement de données
  • lœrn ne sait pas avec qui vous partagez des données
  • Une fois partagées, les données sont gérées par l'application, le service ou le destinataire récepteur selon leurs propres pratiques de confidentialité

Vous êtes seul responsable de la manière et de l'endroit où les fichiers exportés sont partagés.

Responsabilité en matière de droits d'auteur et de contenu

lœrn est un outil, pas un fournisseur de contenu. Vous êtes responsable de :

  • Vous assurer que vous avez le droit d'utiliser le contenu importé
  • Respecter les lois sur les droits d'auteur lors du partage de paquets exportés
  • La légalité de tous les matériaux stockés ou partagés à l'aide de l'application

Profils

Support multi-profils

lœrn prend en charge plusieurs profils utilisateur locaux sur un seul appareil :

  • Chaque profil a des données d'apprentissage isolées
  • Les profils sont identifiés uniquement par un nom et une icône que vous choisissez
  • Les données de profil sont stockées localement et jamais transmises par l'application
  • La suppression d'un profil supprime définitivement ses données de l'appareil

Statut premium

Le statut d'achat premium s'applique à tous les profils sur le même appareil et compte de magasin.

Notifications (facultatives)

Rappels d'apprentissage

Vous pouvez éventuellement activer les rappels d'apprentissage :

  • Les rappels sont programmés localement sur votre appareil
  • Aucun serveur externe n'est utilisé pour la livraison des rappels
  • Vous pouvez désactiver les rappels à tout moment
  • Les autorisations de notification peuvent être révoquées dans les paramètres de l'appareil

Autorisations

L'application peut demander les autorisations suivantes :

Stockage (Android)

  • Objectif : Importation et exportation de fichiers de paquets
  • Portée : Uniquement les fichiers que vous sélectionnez explicitement

Internet (version gratuite uniquement)

  • Objectif : Affichage de publicités
  • Non utilisé : Version premium

Notifications (facultatives)

  • Objectif : Rappels d'apprentissage
  • Contrôle : Vous choisissez la fréquence et le moment

Services tiers

Publicité (version gratuite uniquement)

La version gratuite utilise Google AdMob.

Nous n'utilisons pas :

  • Services d'analyse
  • Services de rapport de crash
  • Fournisseurs de stockage cloud
  • Intégrations de médias sociaux
  • SDK de suivi tiers

Confidentialité des enfants

lœrn est destiné aux étudiants universitaires et aux apprenants plus âgés. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans (ou l'âge applicable dans votre juridiction).

Parce que lœrn ne collecte pas de données personnelles, il est conforme aux réglementations sur la confidentialité des enfants telles que COPPA.

Sécurité des données

Sécurité de l'appareil local

Vos données sont protégées par :

  • Chiffrement au niveau de l'appareil
  • Sandbox d'application
  • Les mécanismes d'authentification de votre appareil

Aucun risque de serveur

Parce que lœrn ne stocke pas de données sur des serveurs, il n'y a aucun risque de :

  • Violations de serveur
  • Fuites de données cloud
  • Accès non autorisé de tiers

Votre responsabilité

Vous êtes responsable de :

  • Sécuriser votre appareil
  • Gérer les sauvegardes de l'appareil (iCloud / sauvegarde Google)
  • Stocker en toute sécurité les fichiers exportés

Vos droits et contrôle

Vous avez un contrôle total sur vos données :

  • Toutes les données d'apprentissage sont stockées localement
  • Vous pouvez exporter des données à tout moment
  • Vous pouvez supprimer des paquets ou des profils
  • La désinstallation de l'application supprime toutes les données locales

Conservation des données

Sur l'appareil uniquement

Les données restent sur votre appareil jusqu'à ce que vous :

  • Supprimiez du contenu ou des profils
  • Effaciez les données de l'application
  • Désinstalliez l'application

Aucune conservation sur serveur

Nous ne conservons pas de données sur des serveurs car nous n'en exploitons aucun.

Sauvegardes de l'appareil

Si activé, votre système d'exploitation peut sauvegarder les données de l'application selon ses propres politiques. Ceci est contrôlé par Apple ou Google, et non par lœrn.

Base juridique (RGPD)

Pour les utilisateurs dans l'EEE :

  • Nous ne traitons pas de données personnelles en tant que responsable du traitement
  • La publicité (version gratuite) est basée sur un intérêt légitime
  • Vous pouvez éviter la publicité en achetant la version premium

Droits de confidentialité en Californie (CCPA)

  • Nous ne vendons pas d'informations personnelles
  • Nous ne partageons pas d'informations personnelles à des fins de marketing
  • La plupart des obligations CCPA ne s'appliquent pas en raison de l'absence de collecte de données

Transferts internationaux de données

Nous ne transférons pas de données d'apprentissage à l'international car :

  • Les données sont stockées localement sur votre appareil
  • Nous n'exploitons pas de serveurs
  • Les données quittent votre appareil uniquement par partage initié par l'utilisateur

Modifications de cette déclaration

Nous pouvons mettre à jour cette déclaration pour refléter les changements d'application ou juridiques.

Nous informerons les utilisateurs en :

  • Mettant à jour la date « Dernière mise à jour »
  • Publiant la déclaration mise à jour
  • Affichant un avis dans l'application pour les changements importants
