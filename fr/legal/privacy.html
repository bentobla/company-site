Collecte de données sur ce site web

Ce site web est un site statique hébergé sur GitHub Pages. Nous ne collectons, ne stockons ni ne traitons de données personnelles via ce site web.

Nous n'utilisons pas de cookies, d'outils d'analyse ni de mécanismes de suivi.

Fichiers journaux du serveur

Le fournisseur d'hébergement (GitHub) peut automatiquement collecter et stocker des informations dans les fichiers journaux du serveur. Cela peut inclure:

  • Type et version du navigateur
  • Système d'exploitation
  • URL de référence
  • Nom d'hôte de l'ordinateur accédant
  • Heure de la demande du serveur
  • Adresse IP

Ces données ne sont pas combinées avec d'autres sources de données et sont stockées par GitHub selon leur politique de confidentialité.

Contact
